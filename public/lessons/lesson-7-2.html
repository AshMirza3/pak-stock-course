Using Online Trading Platforms

Most brokers offer an online portal—desktop and mobile apps—for placing orders. Examples:

  • Lakson Investments (Lakson Online):
    • User-friendly dashboard with real-time quotes, watchlists, and order book view.
    • Charting tools powered by TradingView embedded widget (candlesticks, indicators).
    • Research tab provides daily reports, sector overviews, and recommended lists.
  • JS Global TraderPlus:
    • Desktop & mobile apps with direct market access (DMA) to PSX.
    • Advanced features: one-click ordering, bracket orders (stop-loss + target), and after-market order entry.
    • Integrated news feed from Reuters & local publications.
  • Empirica (EFM):
    • Cloud-based web platform—no installations required.
    • Real-time price alerts via SMS/email, customizable watchlists.

How to place an order online:

  1. Log in with your client code and password.
  2. Select “Buy” or “Sell,” enter ticker and quantity.
  3. Choose order type (Market, Limit). For Limit, specify limit price.
  4. Review summary including brokerage, SECP fee, exchange fee, CDC fee, VAT.
  5. Click “Submit.” You’ll get a real-time execution confirmation if price conditions are met.

Tip: Save your favorite tickers in “Watchlist” to monitor price movements and set automated price alerts.
