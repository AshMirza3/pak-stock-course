Building a Watchlist & Custom Screener (Demo)

As a frontend-only exercise, let’s create a simple “watchlist” demonstration using static HTML/JavaScript. You can adapt this idea later into a React component.

Watchlist Requirements:

  • Allow the user to type a PSX ticker (e.g., HBL, MCB).
  • On press of “Add,” append the ticker to a list below. (In a real app, you’d fetch live prices via an API, but here we use placeholders.)
  • Allow removal of any ticker from the list.

Static Demo Code (for reference only; not executed here):

<div>
  <input id="tickerInput" type="text" placeholder="Enter PSX Ticker (e.g., HBL)" class="border p-2 rounded" />
  <button id="addBtn" class="bg-indigo-600 text-white px-4 py-2 rounded ml-2">Add</button>
  <ul id="watchlist" class="mt-4 space-y-2"></ul>
</div>

<script>
  const addBtn = document.getElementById("addBtn");
  const tickerInput = document.getElementById("tickerInput");
  const watchlist = document.getElementById("watchlist");

  addBtn.addEventListener("click", () => {
    const ticker = tickerInput.value.trim().toUpperCase();
    if (!ticker) return;
    const li = document.createElement("li");
    li.className = "flex justify-between items-center bg-white p-2 rounded shadow";
    li.innerHTML = `
      <span class="font-medium">${ticker}</span>
      <button class="text-red-600 removeBtn">Remove</button>
    `;
    watchlist.appendChild(li);
    tickerInput.value = "";

    li.querySelector(".removeBtn").addEventListener("click", () => {
      li.remove();
    });
  });
</script>

In a real Next.js site, you’d place this inside a React component, manage state with `useState`, and (optionally) fetch live quotes from an API like Alpha Vantage or PSX’s official feed.
